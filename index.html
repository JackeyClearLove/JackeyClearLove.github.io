代理模式 ：通过代理，控制对 对象的访问
初学者对这个定义可能会产生模糊或者混淆，现在我把我的理解与大家分享

鞋厂销售就相当于 鞋厂生产厂商的代理 ，鞋厂只需要关注生产鞋子，卖鞋子的问题就可以交给代理，代理把从鞋厂的鞋子拿出去给顾客销售， 这样，鞋厂 就通过 代理（鞋厂销售），完成了对 顾客（访问对象）的控制
这只是一个小列子，可能会有更加简介清晰的列子，学习的人，可以借助这个案列去理解代理模式，而不要依赖这个列子或者死记硬背，每个人都有自己的学习方法，自己总结并理解的东西，才是自己的。

GOF23-结构型模式-适配器模式
结构型模式 ：适配器模式 ，代理模式 ， 组合模式，桥接模式，外观模式，享元模式
结构型模式的特点：从程序的结构上实现松耦合，从而扩大整体的类，用来解决更大的问题

定义 ：把一个类的接口转换为客户所需要的接口，适配器把原来不兼容的接口转换为兼容的接口，把原本由于接口不兼容而不能一起工作的哪些类可以一起工作。
模式中的角色
目标接口：客户所期待的接口，目标接口可以是具体或者抽象的类，也可以是接口。
需要适配的类 ： 需要适配的类或是适配者类
适配器 ：通过包装一个需要适配的类，把原接口转换为目标接口

socket

By Jackey

發表於 2017-11-06

socket :插座
在各个语言中，网络编程都被称为socket编程
网络分层 ： 应用层–传输层 –网络层–数据链路层
tcp(translation control protocol) 在不可靠的网络中提供可靠的网络协议(协议是，相当于中国说汉语，那么汉语就是中国人的协议，那么英国人交谈的协议就是英语)。
可靠 ：比如打电话 ， A与B建立连接 A：喂喂喂。必须要听到B的回话，否则A就一直喂喂喂，直到B回话，才对B说想要说的内容， 可靠就是建立连接之后，我给你发送数据包，你必须回应我你收到了，我才给你发送下一个。
UDP（user data protocol） 不可靠的网络协议
不可靠 ： 就是我只负责发送包，不管你是否收到，反正我只负责发送。
场景： 比如我们玩英雄联盟，我的伊泽瑞尔发出一个Q，但是因为网络延迟的原因，我的Q技能本来是0.5秒就应该命中敌人，但是却用了3秒，敌人本来就是残血，但是因为网络延迟，没有看到你的技能，还在找位置对你输出，3秒后你的技能击中敌人，你莫名其妙的杀死了对方。这就是用UDP传输的，在UDP传输的过程中，偶尔出现丢包，也是没有多大问题的。
安全性 tcp > udp 传输速度 ： udp > tcp

端口 ：在一台计算机中，最多可以有65536个端口，在设置端口的时候，最好设置在1024以上的端口，因为1024以下的端口可能被系统征用。

bootstarp.jar //tomcat内核
startup.bat startup.sh shutdown.bat shutdown.sh分别我iwindows和linux下的启动批处理文件
批处理：在.bat的文件里面编写的一些windows的命令（跳转，等），以后只要执行一次.bat文件,就可以把里面的内容全部执行完成而不用逐个的执行每个命令

  1. echo off -> 执行命令的时候，不要在命令窗口显示命令本身 单句的用@也可以
    比如
    @dir
    @ping localhost
    等于
    @echo off
    dir
    ping localhost
    2.if[NOT] ERRORLEVEL number command 如果最后运行的程序返回一个等于或大于指定数字的退出代码，则执行后面的命令
    if[NOT] string1==string2 command 如果指定的字符串匹配，那么就执行后面的命令
    if[NOT] EXIST filename command 如果filename存在 就执行后面的命令
    比如： if “%OS%”==“Windows_NT” setlocal
    意思是如果OS等于Windows_NT的话，就执行setlocal命令
    3.rem命令
    对命令的注释或者对命令的功能的注释，以便于阅读，被rem注释的命令批处理文件不会运行
    4.echo %0 %1 %2 表示命令行想批处理文件传递参数 %0 %1 %2分别代表这第1，2，3个参数
    比如： echo aa bb cc 执行后为 aa bb cc 一直可以到9%
    那么要取第十个怎么办？ 我们可以利用shift命令
    shift表示把参数后移一个位置 那么0%就可以代表1%的参数，以此推断，9%就可以代表第十个参数，还要秋后面的参数，可以继续执行shift命令
    shift
    echo %0 %1 %2
    6.goto label 跳转到 label
    在下面定义 :label 标签名随便定义 跳转后执行:label ，程序将执行下一行的命令
