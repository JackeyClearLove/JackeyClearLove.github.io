java后端面试
校招面试面试官注重的点：
1.基础
2.编程能力

1.String StringBuffer和StringBuilder的区别
(1)在执行速度上，String buffer
其实这是JVM耍的一个把戏 String s = “this is” + “only a” + “fun”;就等于
String s = “this is only a fun”;
（2）线程安全
StringBuffer:线程安全 StringBuilder:线程不安全
虽然StringBuilder的速度比StringBuffer快，但是JVM不能保证StringBuilder在字符串缓冲区操作是线程安全的，但是大部分单线程下操作字符串缓冲区可以使用StringBuilder

总结：对于字符串不频繁的操作使用：String
单线程下操作字符串缓冲区使用;StringBuilder
多线程下操作字符串缓冲区使用：StringBuffer
