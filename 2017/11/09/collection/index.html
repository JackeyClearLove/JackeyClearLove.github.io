2.模仿JDK的LinkedList来简单的理解LinkedList的底层
现在采用双向链表来作为LinkedList的链表实现
在攥写LinkedList之前，必须先理解双向链表(DoubleList)
双向链表的一个元素由三个部分组成，
1.指向上一个元素 2.值 3.指向下一个元素
双向链表特殊的节点(Node)为第一个和最后一个
第一个节点的第一部分为空
最后一个节点的最后一个部分为空
下面用代码帮助理解链表以及LinkedList

//首先定义一个Node类，表示LinkedList中的节点
public class Node{
