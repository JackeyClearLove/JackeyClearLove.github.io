JQuery事件机制
在JQuery01中讲了jquery选择器，jquery操作dom,那么这讲来看看jquery的事件机制，jquery动画，jquery的与ajax的使用，jquery插件的使用。

#JQuery事件机制
JQuery在事件写法上，和传统的js有些异同，JQuery往往不再时件名前面家 ‘on’字
如：onClick,onblur,onfocus,onunload,ondbclick,onmouseover,onkeypress等
JQuery统统去掉’on’

JQuery在传统的事件上新加事件如hover,toogle bind…

#JQuery动画

##隐藏与显示
JQuery中，对于元素的隐藏和显示添加了动画效果
show()表示让某个元素显示 ，可加参数
show(‘slow’,’normal’,’fast’)表示显示速度快慢的三种方式，或者指定毫秒显示，如：show(2000);表示2秒逐渐显示出来
hide()表示让某个参数隐藏，同样也有参数控制隐藏的快慢，毫秒数。

##toggle
toogle(毫秒数,函数一，函数二);

##淡入淡出
JQuery实现淡入淡出或者调整透明度的效果非常方便，三个方法供我们选择使用。
fadeIn(arg) //注：arg表示淡入淡出所需要的时间（毫秒数）
fadeOut(arg)
fadeTo(arg1,arg2);
注：arg1表示所需的时间（毫秒数），arg2表示透明度（0-1之间的小数）

##滑动效果
JQuery实现滑入滑出的效果非常方便，亦有三个方法供我们使用
slidDown(ms);
slidUp(ms);
slidToggle(ms);
注：同样也可以设置ms（毫秒数）

#JQuery与ajax
JQuery完美的结合了Ajax技术，而且增加了许多细节，这样我们利用JQuery去实现异步交互会非常的方便，快捷，好用。
JQuery中也给我们提供了对于获取类型文件的处理方式，即：
-全局函数 ：$.get(url,[data],[callback],[type]);
-全局函数： $.post(url,[data],[callback],[type]);
-对于第四个参数type表示文件类型，如html,xml,js,json,text等。
-这个方法在JQuery开发中经常采用
-当然最常用的还是ajax()方法

$(function(){

        $('#btn').click(function(){
            //get方式提交表单:
            //三个参数 1:url 2:参数 3:callback 
            var datas = $('#myform').serialize();
            /*
            $.get('TestServlet?method=get' ,datas , function(data){
                    alert(data);
            });
            */
            /*
            $.post('TestServlet?method=post' ,datas , function(data){
                    alert(data);
            });
            */ 

            // ajax方法 options (json)
            $.ajax({
                    type:'post' ,
                    url:'TestServlet?method=ajax' , 
                    async: true , //异步 
                    cache: false , //清除缓存  ,
                    //timeout:3000 ,
                    data:{
                        username: $('#myform').find('input[name=username]').val(),
                        password: $('#myform').find('input[name=password]').val(),
                        id: $('#myform').find('input[name=id]').val()
                    } ,
                    dataType:'text' ,   //json
                    success:function(data){
                            var data = $.parseJSON(data);
                            alert(data.message);
                    }
            });
        });

});

以上是JQuery结合ajax的范例，基本上get,post用的很少，JQuery.ajax()普遍用的比较多。
html页面就是一个表单中放了两个input,分别为uname和password
